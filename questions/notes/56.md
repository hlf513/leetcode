# 56. 合并区间

```
输入: [[1,3],[2,6],[8,10],[15,18]]
输出: [[1,6],[8,10],[15,18]]
解释: 区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].
```

| 解法        | 思路                                                         | 时间  | 空间 |
| ----------- | ------------------------------------------------------------ | ----- | ---- |
| 排序 + 遍历 | 排序：快速排序(相等则需要判断 end 值)<br />遍历：更新 end，要取最大值 | nlogn | n    |
