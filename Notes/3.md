# 3. 无重复字符的最长子串

```
// "abcabcbb" => "abc"
```

| 解法         | 思路                                                         | 时间复杂度 |
| ------------ | ------------------------------------------------------------ | ---------- |
| 暴力法       | 1. 获取所有子串<br />2. 检查子串是否无重复字符               | n^3        |
| 滑动窗口     | 两个指针 + 集合<br />1. 若右值在集合里：右指针+1（右值写入集合）；计算最大长度<br />2. 若右值不在集合里：左指针+1（元素删除） | 2n         |
| 滑动窗口优化 | 若右值不在集合，左指针 + j                                   | n          |
