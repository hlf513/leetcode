# 60. 第 K 个排列

```
当 n = 3 时, 所有排列如下：

"123"
"132"
"213"
"231"
"312"
"321"

输入: n = 3, k = 3
输出: "213"
```

| 解法       | 思路                                                         | 时间 | 空间 |
| ---------- | ------------------------------------------------------------ | ---- | ---- |
| 按位找规律 | 核心思想：把字符串分组，k/(n-1)! 表示在第几组的第几位<br /><br />核心公式：<br /> - index =  k / (n-1)!  求字符串的下标 <br /> - k' = k % (n-1)! 更新最新字符串的 k 值<br /><br />注意：因为下标从0开始，k 需要初始化为 k-1 | n    | n    |
