# 15. 三数之和

```
给定数组 nums = [-1, 0, 1, 2, -1, -4]，

满足要求的三元组集合为：
[
  [-1, 0, 1],
  [-1, -1, 2]
]
```

难点：不重复

| 解法             | 思路                                                         | 时间  | 空间 |
| ---------------- | ------------------------------------------------------------ | ----- | ---- |
| 排序+三指针+剪枝 | **排序 + 使用三个指针 + 剪枝**<br />1. 固定一个a，用左右指针找和为-a的两个数<br />2. 注意三个指针都要去重<br />3. 注意边界 | nlogn | n    |
