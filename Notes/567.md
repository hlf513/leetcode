# 567. 字符串排列

```
输入: s1 = "ab" s2 = "eidbaooo"
输出: True
解释: s2 包含 s1 的排列之一 ("ba").
```

| 解法         | 思路                                                         | 时间复杂度 | 空间复杂度 |
| ------------ | ------------------------------------------------------------ | ---------- | ---------- |
| 字符出现次数 | 1. 把 s2 按照 s1 长度切分为若干个子串<br />2. 比较s2子串中字符出现次数是否和 s1相等 | O(s1*s2)   | O(s1)      |
