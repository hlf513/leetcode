# 1. 两数之和

```
给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]
```

| 解法      | 思路                                                         | 时间 | 空间 |
| --------- | ------------------------------------------------------------ | ---- | ---- |
| 暴力法    | 遍历两次                                                     | n^2  | 2n   |
| 一遍 hash | 只遍历一次<br />1. 把遍历过的值存在 hash 表中<br />2. 若 target 值在 hash 表中，则直接返回 | n    | 2n   |
